如何用Octopress创建blog

Nov 14th, 2012 | Comments

Install Octopress

install rvm

先安装rvm环境

bash < <(curl -s https://rvm.beginrescueend.com/install/rvm)

將 rvm 指令变成 shell function

echo '[[ -s "$HOME/.rvm/scripts/rvm" ]] && . "$HOME/.rvm/scripts/rvm" # Load RVM function' >> ~/.bash_profile
source ~/.bash_profile

安裝 Ruby 1.9.2, RubyGems

确认版本

ruby --version

Down octopress

git clone git://github.com/imathis/octopress.git octopress
cd octopress

安裝 bundler

gem install bundler
bundle install

安装预设场景

rake install

生成模版

rake generate

本地预览

rake preview

可以通过 http://127.0.0.1:4000 查看生成的blog

添加文章和page页基本操作

添加新文章

rake new_post["文章标题"] //可以直接输入中文，Octopress 自动将汉字转换成拼音url
例如：rake new_post["hello"]

会在source/_post/生成hello.markdown的文件

添加新页面

rake new_page["about"]

其他的编辑项

blog的配置信息修改: _config.yml
